DatabaseHelper

DatabaseHelper class

Esta clase sirve maneja la inicialización y administración de la base de datos en la aplicación, proporcionando métodos para manejar la base de datos tanto en dispositivos móviles como en entornos de escritorio (usando SQLite).

Se adapta a las plataformas Android, iOS y Desktop (usando sqflite-ffi para escritorio).
