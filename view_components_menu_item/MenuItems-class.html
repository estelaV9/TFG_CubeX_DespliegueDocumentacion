MenuItems

MenuItems class

Clase que agrupa los ítems de menú para las penalizaciones.

Esta clase contiene una lista de ítems predefinidos:

  • "No Penalty".
  • "DNF".
  • "+2". Maneja la lógica de actualización del icono de penalización en función del ítem seleccionado. Además, puede construir un Widget para cada item del menú y un método para manejar la selección de penalizaciones.
