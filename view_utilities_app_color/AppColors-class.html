AppColors class

Clase que contiene la paleta de colores utilizada en la aplicación.

Esta clase define constantes de color estáticas que se usan para decorar toda la interfaz de usuario, facilitando así el esquema de colores. Los colores están definidos mediante valores hexadecimales.
