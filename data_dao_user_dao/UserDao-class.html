UserDao class

Clase encargada de gestionar las operaciones CRUD sobre los usuarios.

Esta clase permite validar, insertar, eliminar y verificar la existencia de usuarios. Utiliza la clase DatabaseHelper para realizar las consultas y operaciones necesarias sobre la base de datos.
